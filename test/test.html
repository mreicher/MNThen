Help Guide

Getting Started

Click "I Agree" to accept the safety terms and access the interactive map with historical markers across Minnesota.

Using the Map

Our interactive map displays historical markers throughout Minnesota. Each marker is color-coded by category (see Color Key for details).

Exploring Sites

Tap any marker to view detailed information about that historical site, including photos, descriptions, and location summaries.
