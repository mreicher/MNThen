Marker Categories

Each marker color represents a different category of historical significance:

Help Guide

Getting Started

Tap "Continue" to access the interactive map featuring historical markers across Minnesota.

Using the Map

Our map displays historical markers throughout the state. Each marker is color-coded by category (see Color Key for details).

Exploring Sites

Tap any marker to view detailed information about that historical site, including photos, descriptions, and location summaries.

Desktop & Laptop Users

This app is optimized for mobile phones and tablets. Location check-in, automatic map following, and audio cues rely on mobile sensors and may not function on desktop or laptop browsers. For the complete experience, please use a mobile device.
