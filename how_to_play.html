How to Play

Welcome to the Trivia Game! Here's how to play:

  1. Read the trivia question carefully.
  2. Choose the correct answer from the options provided.
  3. If you select the correct answer, it will be highlighted in green.
  4. If you select the incorrect answer, it will be highlighted in red. The correct answer will also be displayed in green.
  5. Your score will increase for each correct answer.
  6. Try to answer as many questions correctly as possible and climb up the leaderboard!

Things to Know

  • Each game is ten questions with a short break mid-way.
  • Answer five questions in a row to start receiving double points.
  • Make sure to create an account and log in to save your score and get on the All-Time Points leaderboard.
  • The trivia games can be played without creating an account. You just can't save your score.
